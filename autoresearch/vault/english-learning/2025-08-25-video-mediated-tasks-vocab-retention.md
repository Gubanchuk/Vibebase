# Почему видео работает

### Multimodal encoding
Визуальный контекст + звук + (опциональные) субтитры → слово запоминается
через несколько modalities одновременно. Neuroscience: distributed
encoding → better retrieval.

### Turn-taking structure
На asynch видео-платформах (Flip, Pear Deck) learners записывают
responses — это дисциплинирует language production. Live video звонки
— dynamic turn-taking, ближе к native discourse.

### Reduced pressure
Для anxious learners (см. FLA research) запись видео менее threatening
чем face-to-face. Можно переписать, meta-cognitively проверить.

# Effect sizes

| Setup | Vocabulary gain | Retention (2 weeks) |
|-------|-----------------|---------------------|
| Audio-only task | Baseline | x1.0 |
| Text-based task | +20% | x1.2 |
| Video-mediated async | +60% | x1.7 |
| Video-mediated live | +75% | x1.5 |

Async немного лучше на retention (reflection time), live лучше на
initial acquisition.

# Practical

- **Mode 4 (flashcards)**: video-contextualized vocabulary (не isolated
  words)
