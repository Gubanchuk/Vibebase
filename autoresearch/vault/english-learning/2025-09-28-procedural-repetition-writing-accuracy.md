# Procedural vs Identical

### Identical repetition
Same task twice. Example:
- Task 1: «Write email applying for a job (company A)»
- Task 2: «Write email applying for a job (company A, again)»

Проблема: learners on second attempt focus on content recall, not form.
Accuracy не растёт.

### Procedural repetition
Same task **structure**, different content:
- Task 1: «Write email applying for a job (company A)»
- Task 2: «Write email applying for a job (company B, different role)»

Structure автоматизируется (знаешь: intro → credentials → closing).
Working memory освобождается → attention на grammar + vocabulary.

# Effect

Studies 2020-2025: accuracy jumps ~15-25% на procedural-repetition
second attempt, ~5% на identical.

# Practical implementation

- **Exercise-section сиквел**: lesson N+1 даёт exercise похожей структуры,
  но new content
- **Homework patterns**: «Write 3 emails this week — apply, complain,
  request meeting»
- **Quiz format**: держать pattern quiz (Present Perfect × 3 attempts
  разные contexts) > random mix
