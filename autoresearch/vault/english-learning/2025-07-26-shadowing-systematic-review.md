# Что такое shadowing конкретно

Не повторение «после паузы». А **simultaneous** повторение вслух — с
задержкой в миллисекунды. Физически это заставляет speech-motor system
синхронизироваться с target-speaker.

# Что улучшается (по research)

- **Prosody** (ритм, интонация) — самый большой effect size
- **Weak forms** — redactions типа «gonna», «wanna», «shoulda» становятся
  естественными
- **Pitch range** — расширяется под native-norms
- **Comprehensibility** (понятность для native listener) — значимо растёт
- **Accentedness** уменьшается (но не исчезает)

# Practical implementation

1. **Material**: 1-2 минуты native audio (подкаст, TED, видео)
2. **Phase 1** — listening only, понимаем meaning
3. **Phase 2** — shadowing с текстом (читаешь + повторяешь)
4. **Phase 3** — shadowing БЕЗ текста (sheer voice mimicking)
5. **10-15 минут интенсивно**, потом stop — усталость убивает benefit

# Gotcha для русских

Английский — stress-timed, русский — syllable-timed. Без **explicit
phonics work** shadowing может зафиксировать неверные stress-патерны.
Сначала разбери weak forms и schwa-reduction осознанно, потом shadow.

# Применимо у нас

В Mode 2 (lesson) по pronunciation-топикам: audio-exercise с phase 2+3.
Нужна будет audio-инфраструктура (TTS OpenAI или ElevenLabs) — но это v2.
