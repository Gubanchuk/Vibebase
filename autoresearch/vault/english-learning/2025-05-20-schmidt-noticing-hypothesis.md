# Три аспекта consciousness

1. **Awareness** — замечаешь что услышал/прочитал
2. **Intention** — хочешь это запомнить / использовать
3. **Knowledge** — хранишь для извлечения

Для перехода input → intake нужна как минимум awareness. Без неё input
проходит мимо когнитивной системы.

# Не требует full conceptual understanding

Schmidt не требует чтобы learner **понимал правило** — достаточно
notice что форма существует. «А, они говорят *had eaten*, не *ate*»
— это уже noticing, даже без объяснения Past Perfect.

# Criticism (Nick Ellis)

Schmidt misconstrues implicit learning. Повторное exposure to formulaic
sequences **без** focal awareness может привести к retention. Формулы
типа «How are you?» усваиваются без noticing. Но для non-formulaic
grammar — noticing критичен.

# Practical implementation

### Для тьютора:
- **Highlight форму в explanation** — «Обрати внимание: *have been*,
  не просто *been*»
- **Contrastive pairs** — показывай две формы рядом, learner сам замечает
  разницу
- **Enhanced input** — bold, highlight target structures в текстах

### Для quiz:
Вопросы должны требовать discrimination между формами — что тренирует
noticing.

# Применимо у нас

Наш Teacher в concept-секции должен явно «обращать внимание» на целевую
форму. В example-секции дать два варианта (правильный + wrong) и
подсветить разницу. В quiz — контрастивные пары.
