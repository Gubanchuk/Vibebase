# Что такое TBLT

Обучение через **meaningful tasks**, где язык — средство, не цель.
Task = activity с non-linguistic outcome:
- «Solve puzzle with partner using only English»
- «Plan trip to Tbilisi, justify budget»
- «Interview 3 classmates, find common hobby»

Формальный grammar-input идёт ПОСЛЕ задачи, как reflection — не до.

# Research findings 2025

### Procedural repetition > Identical repetition
Повторение **похожей** задачи (не той же) даёт больший прирост accuracy.
Например: первый task «spot the difference» на картинке A, второй — на
картинке B. Дизайн структуры повторяется, контент новый.

### Video-mediated tasks дают extra
- Multimodal input (visual + auditory) → vocabulary retention x1.7
- Structured turn-taking via platform features
- Reduced social pressure (не лицом к лицу)

### GenAI integration (новое в 2025)
GenAI сам генерит task materials под уровень, role-play partners, feedback.
Комбинация даёт scale TBLT one-on-one tutor-style.

# Practical для AI-тьютора

- **Exercise-section** должна быть task-based, не fill-in-blank:
  «Describe your weekend using 3 of these phrasal verbs: X, Y, Z»
- **Procedural repetition** — exercise в следующем уроке похож на
  предыдущий по структуре, но с новым контентом
- **Video-based mode** — v2: подкаст-clip + task «summarize in 30 sec»
