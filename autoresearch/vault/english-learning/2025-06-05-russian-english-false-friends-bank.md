# Top false friends for Russian speakers

| Russian cognate-word | English meaning | Common mistake |
|---------------------|-----------------|----------------|
| actual (актуальный) | factual / real | "actual topic" → use "relevant" |
| sympathetic (симпатичный) | compassionate | "he's sympathetic" → use "likeable" |
| eventually (эвентуально) | finally, in the end | NOT "возможно" |
| fabric (фабрика) | cloth material | NOT "factory" |
| control (контролировать) | to have power over | NOT "check" |
| magazine (магазин) | publication, periodical | NOT "shop" |
| family (фамилия) | whole household | NOT just last name |
| intelligent (интеллигентный) | smart | NOT "cultured/polite" |
| accurate (аккуратный) | precise | NOT "tidy" |
| prospect (проспект) | likelihood, future outlook | NOT "avenue" |
| dramatic (драматический) | significant, intense | NOT "theatrical" only |
| principal (принципиальный) | main, chief | NOT "on principle" |

# Stakes

False friend errors — high-stakes:
1. They often **change meaning** entirely (not just sound non-native)
2. They **reveal L1 background** immediately to native speakers
3. They create **awkward moments** (saying "my sympathetic friend" when
   you mean "attractive")
