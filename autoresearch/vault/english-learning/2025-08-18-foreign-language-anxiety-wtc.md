# Loop FLA → WTC

```
FLA (страх говорить) ↓
   ↓
WTC (желание говорить) ↓
   ↓
Actual L2 use ↓
   ↓
Practice ↓ → Competence ↓
   ↓
FLA ↑ (meta: меньше практики — больше страха)
```

Чтобы разорвать loop — нужна **low-stakes practice**, где цена ошибки
низкая. AI-тьютор идеален: не судит, не спешит, можно пробовать.

# Grit как mediator

У learners с high grit (определение: persistence + passion к long-term
goals) связь FLA→WTC слабее. То есть даже при высокой тревоге они
всё равно говорят, потому что верят в процесс.

**Implication:** мотивация важнее тревоги. Если learner видит progress
— FLA уменьшается автоматически.

# Practical implications для AI-тьютора

1. **Show progress explicitly** — mastery-счётчик, streak, улучшение
   metrics. Виден progress → grit активируется.
2. **Low-stakes framing** — «давай попробуем» vs «ответьте правильно»
3. **No-judgment mode** — в Mode 1 (chat) mistakes не выделять красным,
   не делать big deal. Мягкая коррекция в конце turn'а, не посреди.
4. **Frequent short sessions** > редких длинных. Attentional budget
   у anxious learners ограничен.
