# Mechanism

```
Frequent L2 use
   ↓
Processing becomes automatic
   ↓
Less working-memory load
   ↓
Cognitive bandwidth freed for communication
   ↓
Less anxiety
   ↓
More willingness to use L2
   ↓
(loop back)
```

# Frequency > Intensity

Experimental studies: daily 15-min sessions outperform weekly 2-hour
sessions on:
- Anxiety reduction (measured via FLCAS scale)
- Fluency (WPM)
- Retention (1-month test)

Reason: habit formation. Daily = hard-wired ritual. Weekly = effortful
event.

# Daily session design

Optimal:
- **Fixed trigger** (e.g. morning coffee) — habit cue
- **15-20 min** — below fatigue threshold
- **Low stakes** — no grades, no judgment
- **Visible progress** — streak counter

Что **не работает**:
- Extensive grammar drills — cognitively exhausting, kills habit
- Perfectionism — forces anxiety
- Skipping allowed — breaks habit

# Practical для AI-тьютора

- **Streak counter** в UI — позитивное подкрепление habit
- **Daily reminder** (opt-in) — через 24 часа если не заходил
- **Short default sessions** — default exercise-length подстраивается
  под usage
- **No hard grading** — soft pass-through на exercises, mastery как
  motivation, не блокер
