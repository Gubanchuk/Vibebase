# Strong vs Weak Noticing

### Strong (Schmidt 1990)
No learning without conscious noticing. Input that's not noticed = just
noise.

### Weak (Schmidt 1994, Ellis 2007)
Noticing helpful but not always required. Implicit learning of formulaic
sequences is possible through frequency alone.

# Non-salient formulas examples

- Weak forms: «gonna», «wanna», «shoulda», «dunno»
- Discourse markers: «you know», «I mean», «kinda»
- Collocations без logic: «make a decision» (not *take*), «take a shower»
  (not *make*)
- Grammatical collocations: «depend ON» (not *from*), «interested IN»
  (not *with*)

Эти не required noticing для pickup — достаточно frequency.

# Practical implication

Two-track learning:
1. **Implicit track** (extensive input, listening, reading) — picks up
   formulas
2. **Explicit track** (focus-on-form, lessons, drilling) — picks up
   non-formulaic grammar

Effective tutor uses both.

# Practical для AI-tutor

- **Mode 1 (chat)**: extensive input — native-like formulas via model
  choice of phrasing. User sees them often, picks up implicitly
- **Mode 5 (grammar)**: explicit track для non-formulaic rules (tenses,
  conditionals)
- **Don't over-explain formulas**: force explanation of *make/take*
  collocations — more confusion than help. Let frequency teach.

# Применимо у нас
