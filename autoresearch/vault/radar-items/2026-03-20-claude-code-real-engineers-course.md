# Почему cohort

Solo-курсы (видео-библиотеки) плохо работают для agent-workflows — нужны
peers чтобы сравнивать подходы и catch anti-patterns рано. Cohort с
office-hours и peer review даёт feedback loop.

# Применимо у нас

Показывает что формат «структурный курс vs chat» — востребованная модель
в community. Подтверждает наш путь на Vibebase. Темы курса можно
рассматривать как reference-list для нашего Curator'а.
