# What to do in our codebase

Our `AiKernel` interface should expose a kernel method that maps to
`ToolLoopAgent` when we add tool calling (Phase 2+). Keep the current
`stream()` method one-shot; add `runAgent()` later with an approval hook
parameter.

Execution-approval integrates with `useChat` — so if we add tools to English
tutor modes (e.g. "lookup a word in dictionary API"), the user can approve
tool calls inline in the chat UI.
